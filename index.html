Information on V-Chat 2.0

V-Chat's system requirements

  • Direct X 3.0
  • Win95 or better
  • 8MB Ram
  • 486 processor or better
  • about 3MB space

V-Chat features

  • Both graphical and text based chatting experiences
  • Save chat histories.
  • Using the Avatar Wizard you can make or customize an avatar.
  • Password your private rooms
  • kick out unwanted or obnoxious guests in your own rooms
  • Emote through avatar action buttons, or the button beside your text box.

V-chat is a 3D chat program that involves the user with other users in rich 3D environments. MSN relinquished it as a service in 2001 and stopped developing the chat.

You can walk,fly,and look up and down through the 'rooms' using your arrow keys (up,down,left,right).You can also use your mouse to move around and click "portals" to navigate to different rooms.

How to 'fly' in V-chat.

  • ctrl+up arrow key,to 'fly' up
  • ctrl+down arrow key, 'fly' down
  • ctrl+left arrow key, 'sidestep' left
  • ctrl+right arrow key, 'sidestep' right

How to look up and down in V-chat.

  • ctrl+alt+up arrow key,to reset your veiw
  • ctrl+alt+down arrow key, to look down
  • ctrl+alt+left arrow key, to look left
  • ctrl+alt+right arrow key, to look right

What's an Avatar?

Avatars are 2d representations of yourself in V-Chat. With the Avatar Wizard you can make your own avatars.

Installing Microsoft V-Chat and how to get it working on Windows 10/11

Please note that if any of the files below trigger windows defender, allow them, defender is falsly identifying non existant trogens when it comes to some older 16 and 32 bit files.

  • First you need to download and install V-Chat
  • After that navigate to C:\Program Files (x86)\Microsoft V-Chat
  • Rename WORLDS to Worlds
  • Rename NEWWOR to NewWorlds
  • Open the Worlds folder and rename HOME to DefHomespace
  • Next Download dgVoodoo2 & d3drm files.
  • Extract the contents to C:\Program Files (x86)\Microsoft V-Chat
  • Now open dgVoodooCpl.exe
  • Make your settings match mine.
  • Now you can run V-Chat as administrator so it can write to the drive and you are good to go!
If you would like to add the ability to see .hlp files as they are meant to be seen Download and extract the zip file, then run install.cmd as administrator. Now you will beable to use the help files included in V-Chat!
