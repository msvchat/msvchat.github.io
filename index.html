    Information on V-Chat 2.0

    V-Chat's ORIGINAL system requirements

    • Direct X 3.0
    • Win95 or better
    • 8MB Ram
    • 486 processor or better
    • about 3MB intial space*
    *V-Chat downloads worlds when you enter an IRC chatroom with a graphical link in its description which looks like this v:{-a https://msvchat.github.io/vchat/worlds/lobby/world.adf}

    V-Chat features

    • Both graphical and text based chatting experiences
    • Save chat histories.
    • Using the Avatar Wizard you can make or customize an avatar.
    • Password your private rooms
    • kick out unwanted or obnoxious guests in your own rooms
    • Emote through avatar action buttons, or the button beside your text box.

    V-chat is a 3D chat program that involves the user with other users in rich 3D environments. MSN relinquished it as a service in 2001 and stopped developing the chat.

    You control your avatar as you walk,fly,and look up and down through the 'rooms' using your arrowkeys (up,down,left,right).You can also use your mouse to move around and click "portals" to navigate to different rooms.

    What's an Avatar?

    Avatars are 2d representations of yourself in V-Chat. With the Avatar Wizard you can make your own avatars.

    How to 'fly' in V-chat.

    • ctrl+up arrow key,to 'fly' up
    • ctrl+down arrow key, 'fly' down
    • ctrl+left arrow key, 'sidestep' left
    • ctrl+right arrow key, 'sidestep' right

    How to look up and down in V-chat.

    • ctrl+alt+up arrow key,to reset your veiw
    • ctrl+alt+down arrow key, to look down
    • ctrl+alt+left arrow key, to look left
    • ctrl+alt+right arrow key, to look right

    Installing Microsoft V-Chat 2.0 and how to get it working on Windows 10/11

    Please note that if any of the files below trigger windows defender, allow them, defender is falsly identifying non existant trogens when it comes to some older 16 and 32 bit files.

    Second note: because the installers are broken for later operating systems you will need something like Geek Uninstallerto forcefully uninstall V-Chat, I am sorry about this. I hope that by repackaging the installer(when I get around to it) I can remedy this issue.

    • First you need to Download Microsoft V-Chat 2.0extract it from the zip and install it(github gives warnings for hosting an exe file unless its a release so I zipped it to solve this issue)
    • After you've done that, navigate to C:\Program Files (x86)\Microsoft V-Chat
    • Rename WORLDSto Worlds
    • Rename NEWWORto NewWorlds
    • Open the Worldsfolder and rename HOMEto DefHomespace
    • Next Download dgVoodoo2 & d3drmfiles.
    • Extract the contents to C:\Program Files (x86)\Microsoft V-Chat
    • Now open dgVoodooCpl.exe
    • Make your settings match mine.
    • Right click on VCHAT20.EXEchoose properties
    • Click on the "Compatibility" tab, check the box next to "Run this program as administrator"
    • Click Apply and then OK
    • Now you can run V-Chat!

    IT IS HIGHLY RECOMMENDED...that you add the ability to see .hlp files as they are meant to be seen!

    Why? Because most of V-Chat and its related utilities useful knowledge is locked away behind these files. Plus it is pretty cool to beable to use the help function like it was meant to be used.
    Download and extract the zip file, then run install.cmd as administrator. Now you will beable to use the help files included in V-Chat!

    NOTE

    There is a bug with V-Chat on modern hardware that occurs when you have the "graphical window" selected and you begin to type. Often you will get a spam of the first key you hit after that the text box has been auto selected and you will begin to type normally.To avoid this just click the text box before you plan to type in chat.
