Installing Microsoft V-Chat 2.0 and how to get it working on Windows 10/11

Please note that if any of the files below trigger windows defender, allow them, defender is falsely identifying nonexistent trogens when it comes to some older 16 and 32 bit files.

Second note: because the installers are broken for later operating systems you will need something like Geek Uninstaller to forcefully uninstall V-Chat, I am sorry about this. I hope that by repackaging the installer(when I get around to it) I can remedy this issue.

  • First you need to Download Microsoft V-Chat 2.0 and install it.
  • After you've done that, navigate to C:\Program Files (x86)\Microsoft V-Chat
  • Rename WORLDS to Worlds (this step isn't necessary. It just makes the folder look nicer not being all caps.)
  • Rename NEWWOR to NewWorlds
  • Open the Worlds folder and rename HOME to DefHomespace
  • Next Download dgVoodoo2(2.79.3) & d3drm files.
  • Extract the contents to C:\Program Files (x86)\Microsoft V-Chat
  • Now open dgVoodooCpl.exe
  • Make your settings match these.(The Glide tab is unused by V-Chat in windows operating systems and the settings don't need to be set.)
  • Right click on VCHAT20.EXE choose properties
  • Click on the "Compatibility" tab, check the box next to "Run this program as administrator"
  • Click Apply and then OK
  • Now you can run V-Chat!

IT IS HIGHLY RECOMMENDED...that you add the ability to see *.hlp files as they are meant to be seen!

Why? Because most of V-Chat and its related utilities useful knowledge is locked away behind these files. Plus it is pretty cool to be able to use the help function like it was meant to be used.

Download and extract the zip file, then run install.cmd as administrator. Now you will be able to use the help files included in V-Chat!

NOTE

There is a bug with V-Chat on modern hardware that occurs when you have the "graphical window" selected and you begin to type. Often you will get a spam of the first key you hit after that the text box has been auto selected and you will begin to type normally.To avoid this just click the text box before you plan to type in chat.
