Installing Microsoft Author and getting it to work on Windows 10/11

  • First Install V-Chat
  • Then Download Microsoft V-Chat Author 1.1 and begin the installation, except you need to make sure it installs to "C:\Program Files (x86)\Microsoft V-Chat"
  • After you've done that, navigate to C:\Program Files (x86)\Microsoft V-Chat
  • Right click on AUTHOR11.EXE choose properties
  • Click on the "Compatibility" tab, check the box next to "Run this program as administrator"
  • Click Apply and then OK
  • Now you can run the Microsoft V-Chat Author!
